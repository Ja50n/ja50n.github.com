工具控，不管是Windows，Android，还是新装的Ubuntu，都在寻找提高效率的工具和方法。本文做个记录，方便以后更换设备或系统时快速配置工作环境。目前在使用和使用过的远不止下面这些，一时间想不起来了，以后用到想到会更新。




科学上网
  • 红杏，Chrome/FireFox/Safari/UC/猎豹等浏览器插件
  • 赛风，免费Android/Windows平台软件
  • Shadowsocks，著名代理工具




跨平台同步/推送/协作
  • Pushbullet，支持Android/Chrome/FireFox等
  • 1password，密码管理工具，支持Mac/Windows/iOS/Android平台
  • 360云盘，不想用360，但是目前我找不到比360云盘更好用的云盘
  • Tower，团队协作工具，可以在微信公众平台中使用

Ubuntu
  • Unity Tweak Tool
  • 在当前位置打开终端

sudo apt-get install nautilus-open-terminal
nautilus -q

  • Source Code Pro

wget https://github.com/adobe-fonts/source-code-pro/archive/1.017R.zip
unzip 1.017R.zip
mkdir -p ~/.fonts
cp source-code-pro-1.017R/OTF/*.otf ~/.fonts/
fc-cache -f -v

sudo update-pepperflashplugin-nonfree
